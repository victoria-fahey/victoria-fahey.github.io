CSS Concepts - class vs. id

.class

The .class selector allows us to apply the same style to different HTML elements at once.

How it does this is by matching the .class selector name in the CSS file with the element that has the same class attribute value in the HTML.


CSS syntax:

.class {
css declarations;
}

The class selector can be used within an HTML file directly or used in an external CSS file to apply specific styling, and then called in an HTML file. However, for the sake of this blog we will only look at using the class selector with an external css file.

#id

The #id selector is very similar to .class except that it's used for selecting a single HTML element with a unique id attribute value and is not reusable, unlike .class which can be used for multiple HTML elements.

The other difference is that the #id selector uses a hash # in front of the id name.


CSS syntax:

#id {
css declarations;
}
